📖 全新中心化数据驱动使用说明

目前链接已全部升级为远程云端控制，若要开通或修改账号链接，请执行以下两步：

第一步：去 GitHub 配置/更新数据库

  1. 在 GitHub 仓库中创建或编辑同目录下的 users.json 文件。
  2. 按照格式添加用户，必须以 "学号" 作为大括号外层的暗号（键值）。
  3. 如果需要远程封禁或销毁某个链接，直接在 JSON 中把该用户的 "status": "active" 改为 "status": "expired" 即可。

第二步：获取该用户的专属链接

  1. 方法 A（自动生成）：在本控制台上方输入或🎲随机出你想开通的学号资料，点击上方的 生成并复制... 按钮，系统将自动把缩短后的完美暗号网址复制到剪贴板。
  2. 方法 B（手动拼接）：直接在你的主站网址后拼接参数：?id=对应学号 即可直接分发（例如：?id=2016211020208）。

💡 关于样式与字体更新说明：
